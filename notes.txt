What is Database?
- Collection of tables
- Because not all tables share the same structure of data
- Could store information about the claims in one table, could store the information about the insurance plans in another table
- Databases are of two types: SQL (Structured Query Language)
                              NoSQL (Not Only Structured Query Language)
- SQL databases example: MySQL, Postgresql, SQLite, Oracle
- NoSQL databases example: MongoDB, CockroachDB
- SQL databases rely on the concept of relationships
- SQL supports queries
- Queries are the medium through which we interact with the database engine and work with data
- CRUD (Create, Read, Update and Delete) Operations

What is table?
- Collection of data
- Structured data -> same characteristics of data
- Stored in tables
- Rows and Columns
- Columns -> Defines the structure and properties that need to be stored
- Rows -> Data itself
- Schema: It defines the structure of data


What do we learn while studying SQL?
- How to create a database
- How to create a table
- How to add data to a table
- How to read the data from table
- How to update the data in table
- How to delete the data from table
- How to perform calculations and analysis in the data without changing the data itself

Types of Queries:
1. DDL (Data Definition Language) - It is used to define the structure of tables (CREATE)
2. DML (Data Manipulation Language) - It is used to work with the data in the table itself (INSERT, UPDATE, DELETE)
3. DQL (Data Query Language) - It is used to retrieve and read the data from the table (SELECT)

SQL Format:
- It is plain English like statement
- It always ends in semicolon (;)
- It doesnot care about cases (CREATE and create are the same thing)
- Standard Convention for tables is SQL keywords - uppercase, user words - lowercase
Example: CREATE TABLE students;

Data Types in SQL:
1. Numbers: INT, SMALLINT, BIGINT, TINYINT, DECIMAL(p, s)
Example: Decimal(10, 4) - 999999.9999 0.0000
2. String/Character: CHAR(n), VARCHAR(n)
Example: VARCHAR(5) - "Hello"
3. TEXT, TINYTEXT, LONGTEXT
4. DATE, TIME
5. BOOLEAN: TRUE or FALSE
